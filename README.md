# Subdomain Capturer

<h4 align="center">Another automated script to check for subdomain takeover.</h4>

#### Installation
##### build from source
```
git clone https://github.com/Ganofins/subcapture.git && cd subcapture && python3 subcapture -w ./subdomain-list
```
